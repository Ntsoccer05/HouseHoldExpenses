# 複数選択コピー機能 実装計画

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** カレンダービューで日付をクリック → 右パネル（TransactionMenu）の支出カードにチェックボックスを追加し、複数選択して別の日付へ一括コピーできるようにする。

**Architecture:** 右パネルのコンポーネント `TransactionMenu.tsx` に常時表示のチェックボックスと一括コピーボタンを追加する。バルクコピーの API 呼び出しは `TransactionContext.tsx` に `onCopyMultipleTransactions` として追加し、既存のキャッシュ管理（`refreshMonthCache`）と整合させる。バックエンドの `POST /api/copyMultipleContents` は実装済み。

**Tech Stack:** React 18, TypeScript, Material-UI v5, date-fns, Axios（apiClient）

## 設計メモ

### API 呼び出し仕様

```
POST /api/copyMultipleContents
Body: {
  source_date: "YYYY-MM-DD",       // コピー元の日付（currentDay）
  destination_date: "YYYY-MM-DD",   // コピー先の日付（ユーザー入力）
  content_ids: number[]             // Transaction.id を parseInt した配列
}

成功レスポンス: { success: true, message: "2件の支出をコピーしました", copied_count: 2, created_ids: [101, 102] }
エラーレスポンス: { success: false, message: "...", error_code: "SAME_DATE_ERROR" }
```

※ `Transaction.id` はフロントエンドでは `string`。API 呼び出し時に `parseInt(id)` で変換する。

### TransactionMenu.tsx の UI 変更方針

- チェックボックスを常時表示（CardActionArea の外に配置）
- Card をクリック → 既存の編集フォームを開く（変更なし）
- Checkbox をクリック → 選択トグル（編集は起動しない）
- 全選択チェックボックスを「内訳」セクションに追加
- 選択数 > 0 のとき「X件をコピー」ボタンを表示
- 日付変更時（`currentDay` が変わったとき）選択状態をリセット

```
内訳セクション：
┌─────────────────────────────────────┐
│ [☑全選択] 内訳   [2件をコピー]  内訳を追加 │
├─────────────────────────────────────┤
│ [☑] [Card: 食費  スーパー  ¥3,000]  │
│ [☐] [Card: 住居費 家賃    ¥100,000] │
└─────────────────────────────────────┘
```

```typescript
const onCopyMultipleTransactions = useCallback(
    async (
        transactionIds: string[],
        sourceDate: string,
        destinationDate: string
    ) => {
        await apiClient.post("/copyMultipleContents", {
            source_date: sourceDate,
            destination_date: destinationDate,
            content_ids: transactionIds.map((id) => parseInt(id, 10)),
        });

        // コピー元の月とコピー先の月のキャッシュを無効化
        const sourceMonth = format(new Date(sourceDate), "yyyyMM");
        const destinationMonth = format(new Date(destinationDate), "yyyyMM");
        await refreshMonthCache(sourceMonth);
        if (sourceMonth !== destinationMonth) {
            await refreshMonthCache(destinationMonth);
        }
    },
    [refreshMonthCache]
);
```

- [ ] **Step 5: チェックボックス操作ハンドラを追加**

既存の `handleTransactionClick` の後に以下を追加:

```tsx
// 個別チェックボックスのトグル
const handleToggleSelect = useCallback((id: string, e: React.MouseEvent) => {
    e.stopPropagation();
    setSelectedIds((prev) =>
        prev.includes(id) ? prev.filter((sid) => sid !== id) : [...prev, id]
    );
}, []);

// 全選択/全解除
const handleSelectAll = useCallback(() => {
    setSelectedIds((prev) =>
        prev.length === dailyTransactions.length
            ? []
            : dailyTransactions.map((t) => t.id)
    );
}, [dailyTransactions]);

// 一括コピーダイアログを開く
const handleBulkCopyClick = useCallback(() => {
    setBulkCopyDialog({ open: true, destinationDate: '' });
}, []);

// 一括コピー実行
const handleExecuteBulkCopy = useCallback(async () => {
    if (!bulkCopyDialog.destinationDate || selectedIds.length === 0) return;

    if (currentDay === bulkCopyDialog.destinationDate) {
        showSnackBar({
            title: "エラー",
            bodyText: "同じ日付にはコピーできません",
            backgroundColor: "#d32f2f",
        });
        return;
    }

    setOperationState({
        isOperating: true,
        operationType: 'copy',
        message: `${selectedIds.length}件をコピー中...`,
    });

    try {
        await onCopyMultipleTransactions(
            selectedIds,
            currentDay,
            bulkCopyDialog.destinationDate
        );

        showSnackBar({
            title: "コピー完了",
            bodyText: `${selectedIds.length}件を${formatJPDay(bulkCopyDialog.destinationDate)}にコピーしました`,
            backgroundColor: "#455a64",
        });

        setSelectedIds([]);
        setBulkCopyDialog({ open: false, destinationDate: '' });
    } catch (error) {
        console.error("一括コピー失敗:", error);
        showSnackBar({
            title: "エラー",
            bodyText: "コピーに失敗しました",
            backgroundColor: "#d32f2f",
        });
    } finally {
        setOperationState({ isOperating: false, operationType: null, message: '' });
    }
}, [
    selectedIds,
    currentDay,
    bulkCopyDialog.destinationDate,
    onCopyMultipleTransactions,
    showSnackBar,
]);
```
